Available voice commands:

  • Open Options and Close Options - interacts with options window
  • Start Painting and Stop Painting - allows the user to start and stop painting whenever they want.
  • With the options menu, Stroke Style and Color can also be used
Now you only need to move your face to move the brush

Change Color

To change the brush color say "color" followed by one of options below. For a colorful brush, use "color all".

     
Change Stroke

To change the brush stroke say "stroke" followed by one of options below.
